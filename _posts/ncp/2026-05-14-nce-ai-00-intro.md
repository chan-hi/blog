NCE-AI는 네이버 클라우드 플랫폼의 AI 서비스 전반을 다루는 Expert 레벨 자격증이다.
단순한 서비스 소개를 넘어, HyperCLOVA X의 동작 원리와 CLOVA Studio의 기능을 실제 비즈니스 시나리오에 적용할 수 있는 수준을 요구한다.

이 시리즈는 NCE-AI 시험 범위를 주제별로 정리한 학습 노트다.

## NCP 인증 체계

| 등급 | 자격증 | 요구 수준 |
|------|--------|----------|
| Associate | NCA | NCP 기본 이해, 인프라 기초 구성 |
| Professional | NCP | 전문 지식으로 독자적 인프라 구축 |
| Professional AI | NCP-AI | AI 상품 지식 및 적용 능력 |
| **Expert** | **NCE** | 트러블슈팅, 커스터마이징, 심화 운영 |
| **Expert AI** | **NCE-AI** | LLM 이해, CLOVA Studio 전반, RAG/HCX 챗봇 비즈니스 적용 |

NCE-AI는 단순 암기보다 개념 간 연결과 실제 활용 시나리오 이해가 핵심이다.

## 학습 목차

### 1단계: AI 기초

1. AI Overview — 머신러닝, 딥러닝, LLM 개념 정리
2. 네이버 클라우드 AI 서비스 — NCP AI 상품 전체 개요
3. HyperCLOVA X 소개 — 구조, 특징, 성능
4. HyperCLOVA X 구축 사례 — 실제 비즈니스 적용 예시

### 2단계: CLOVA Studio 핵심

5. CLOVA Studio Playground — 기본 사용법, 파라미터 조정
6. 프롬프트 엔지니어링 — Few-shot, Chain-of-Thought, 역할 지정
7. CLOVA Studio 튜닝 — Task Tuning, Domain Tuning, 데이터 준비

### 3단계: 고급 기능

8. CLOVA Studio 익스플로러 — Segmentation, Sliding Window, 토큰 계산
9. RAG 아키텍처 — 검색 증강 생성, 랭체인 연동, PDF 기반 구성
10. 스킬 트레이너 — Skill/Router 연동, Function Calling
11. 이용량 정책 — 토큰 제한, 과금 구조, 운영 시 주의사항

## 시험 포인트 미리보기

- HyperCLOVA X와 일반 LLM의 차이점
- CLOVA Studio 각 기능의 용도와 파라미터 의미
- 프롬프트 엔지니어링 기법의 종류와 특징
- RAG의 동작 원리와 구성 요소
- 이용량 정책의 제한 기준 수치
