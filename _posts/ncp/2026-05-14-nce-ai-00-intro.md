## 핵심 요약

- 본 교재는 네이버클라우드플랫폼 공인 교육 과정인 `NCE-AI`를 위한 가이드입니다.
- 교육은 2일 과정으로 진행되며, HyperCLOVA X와 CLOVA Studio를 중심으로 프롬프트, 튜닝, 익스플로러, RAG, 스킬 트레이너, 이용량 정책을 다룹니다.
- 실습은 교육생 계정과 교육용 크레딧으로 진행되며, 실습 후 리소스 반납이 중요합니다.

## 교육 운영 정보

### 교육 소개

교육에서는 다음 항목을 안내합니다.

- 교육 목적
- 교육 내용
- 교육 시간
- 강사 소개
- 강의장 소개
- 기타 안내

교육 시간은 2일 과정이며, 원문 기준 오전 10시부터 오후 5시까지 진행됩니다.

### 교육용 크레딧

실습을 위해 교육용 크레딧이 제공됩니다.

| 항목 | 내용 |
|---|---|
| 크레딧 금액 | 100,000원 |
| 유효 기간 | 등록 시점부터 한 달 후 23시 59분 59초까지 |
| 안내 방식 | 잔여 금액 50%, 25%, 0% 시 이메일/SMS 안내 |
| 사용 가능 범위 | NCP 실습, NAVER WORKS 사용 가능 |
| 주의 사항 | 실습 리소스를 반납하지 않으면 추가 요금이 청구될 수 있음 |

시험이나 교육 관련 안내는 사전에 이메일로 전달된 문서를 반드시 확인해야 합니다.

## 네이버 클라우드 플랫폼 인증 체계

| 등급 | 의미 | AI 관련 설명 |
|---|---|---|
| NCAssociate, NCA | NCP 기본 이해와 인프라 구성이 가능한 수준 | 기본 이해 중심 |
| NCProfessional, NCP | NCP 전문 지식과 독자적 인프라 구축, 다양한 기능 구현이 가능한 수준 | `NCP-AI`는 NCP 전문 지식과 AI 상품 적용 능력 중심 |
| NCExpert, NCE | NCP 전문 지식, 트러블슈팅, 커스터마이징이 가능한 수준 | `NCE-AI`는 LLM 이해, CLOVA Studio 전반, RAG, HCX 기반 챗봇 구성 등 비즈니스 적용 능력 중심 |

시험 관점에서는 `NCE-AI`가 단순 상품 소개보다 더 높은 수준입니다. LLM 개념을 이해하고, CLOVA Studio의 기능을 실제 비즈니스 시나리오에 연결할 수 있어야 합니다.

## 교육 목차

### 1일차

1. AI Overview
2. 네이버클라우드 AI 서비스
3. HyperCLOVA X 소개
4. HyperCLOVA X 구축 사례
5. 프롬프트 엔지니어링 Lab
6. CLOVA Studio 튜닝
7. Lab: 여성 페르소나 튜닝 모델을 적용한 챗봇 UI 구성

### 2일차

1. CLOVA Studio 익스플로러
2. Lab: 토큰 계산기 API 테스트
3. Lab: Segmentation API를 통한 문단 나누기
4. Lab: Sliding window API를 활용한 챗봇 UI 구성
5. RAG 아키텍처 및 실습
6. Lab: 랭체인을 활용한 PDF 파일 기반 RAG 구성
7. Lab: RAG 데모 사이트 구성
8. CLOVA Studio 스킬 트레이너
9. Lab: 지역 관련 Skill 조회하기 / 라우터 연동하기
10. Lab: 유튜브 영상 추천 Skill 조회하기
11. Lab: Function calling 구성
12. CLOVA Studio 이용량 정책

## 시험 암기 포인트

- `NCE-AI`는 LLM, CLOVA Studio, RAG, HCX 챗봇 구성까지 비즈니스 적용이 가능한 수준을 목표로 합니다.
- 교육 실습 후에는 리소스를 반드시 반납해야 합니다.
- 1일차는 AI 개념, HyperCLOVA X, 프롬프트, 튜닝 중심이고, 2일차는 익스플로러, RAG, 스킬 트레이너, 이용량 정책 중심입니다.

## 원문 반영 체크

- 반영 페이지: 1~8p
- 포함 내용: 문서 정보, 교육 소개, 교육용 크레딧, 인증 제도, 1일차/2일차 목차
