## 핵심 요약

네이버 클라우드 플랫폼은 음성, 번역, 추천, 챗봇, OCR, 이미지 판독, 생성형 AI 개발 도구 등 다양한 AI 서비스를 제공합니다. 시험에서는 상품명과 기능을 연결해서 기억하는 것이 중요합니다.

## AI 상품 정리

| 상품 | 주요 기능 |
|---|---|
| CLOVA Voice | 고품질 음성 합성 기술로 다양하고 자연스러운 목소리 제공 |
| Search Trend | 네이버 통합검색에서 수집되는 검색어 통계를 다양한 기준으로 조회하고 활용 |
| Papago Translation | AI 기반 번역 |
| AITEMS | 사용자별 이력을 분석하여 관심사와 취향에 맞는 상품 추천 |
| CAPTCHA | 다양한 방식으로 사람과 컴퓨터를 판별하여 어뷰징 방지 |
| CLOVA Chatbot | 챗봇 대화 모델 작성과 다양한 메시징 채널 연계 |
| CLOVA Studio | 초대규모 AI HyperCLOVA 기반의 No Code AI 도구 |
| CLOVA OCR | 인쇄물 상의 문자를 인식하고 디지털 데이터를 자동 추출 |
| CLOVA GreenEye | 이미지를 판독하고 유해 이미지를 탐지 |
| CLOVA Speech | 정형화되지 않은 말소리를 인식하여 텍스트로 변환 |
| CLOVA Dubbing | 음성합성 기술로 만든 AI 보이스를 동영상에 입힘 |

## 시험 암기 포인트

- `CLOVA Studio`는 HyperCLOVA 기반 No Code AI 도구입니다.
- `CLOVA OCR`은 문자 인식과 데이터 추출입니다.
- `CLOVA Speech`는 음성을 텍스트로 바꾸는 음성 인식입니다.
- `CLOVA Voice`와 `CLOVA Dubbing`은 모두 음성합성 계열이지만, Dubbing은 동영상에 AI 보이스를 입히는 기능입니다.
- `AITEMS`는 추천, `Search Trend`는 검색어 통계, `CAPTCHA`는 어뷰징 방지입니다.
