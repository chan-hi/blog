## 문제 1

다음 중 네이버 클라우드 플랫폼(MC)에 없는 상품 카테고리는?

**보기:**
- A. Media
- B. Game
- C. Hybrid & Private Cloud
- D. Market Place

> **정답: D. Market Place**
>
> **해설:** NCP의 상품 카테고리에는 Compute, Storage, Database, Networking, Media, Security, Management, Analytics, AI, Dev Tools, Hybrid & Private Cloud 등이 있습니다. Game과 Market Place 카테고리는 존재하지 않습니다.

---

## 문제 2

다음 상품 중 카테고리가 다른 상품은?

**보기:**
- A. GeoLocation
- B. Chatbot
- C. Cloud Outbound Mailer
- D. SourceCommit

> **정답: D. SourceCommit**
>
> **해설:** GeoLocation, Chatbot, Cloud Outbound Mailer는 AI-Application Service 카테고리 상품입니다. SourceCommit은 Dev Tools(개발자 도구) 카테고리의 소스코드 관리 상품입니다.

---

## 문제 3

다음 상품 중 카테고리가 다른 상품은?

**보기:**
- A. Effective Log Search & Analytics
- B. Web Service Monitoring System
- C. Real User Analytics
- D. Cloud Hadoop

> **정답: D. Cloud Hadoop**
>
> **해설:** Effective Log Search & Analytics(CLA), Web Service Monitoring System(WMS), Real User Analytics(RUA)는 Management 카테고리 상품입니다. Cloud Hadoop은 Analytics 카테고리 상품입니다.

## 문제 4

네이버클라우드플랫폼 AI 서비스에서 제공하지 않는 상품은?

**보기:**
- A. Clova OCR
- B. Clova Image Recognition
- C. Clova Voice
- D. Clova Face Recognition

> **정답: B. Clova Image Recognition**
>
> **해설:** NCP AI 서비스에는 Clova OCR, Clova Voice, Clova Face Recognition 등이 있습니다. Clova Image Recognition은 제공되지 않는 상품명입니다.

## 문제 5

현재 네이버 클라우드 플랫폼의 리전으로 제공되지 않는 지역은?

**보기:**
- A. 미국 서부
- B. 미국 동부
- C. 홍콩
- D. 싱가폴

> **정답: B. 미국 동부**
>
> **해설:** NCP는 미국 서부 리전을 제공하지만, 미국 동부 리전은 제공하지 않습니다. 홍콩과 싱가포르는 제공 중인 리전입니다.

## 문제 6

대용량 데이터를 네트워크 전송이 아닌 이동식 스토리지를 이용하여 데이터를 이전하는 NCP 상품은?

**보기:**
- A. Data Teleporter
- B. Data Migration
- C. Data Mover
- D. Data Transfer

> **정답: A. Data Teleporter**
>
> **해설:** Data Teleporter는 물리적 이동식 스토리지 장치를 통해 대용량 데이터를 NCP로 이전하는 서비스입니다. 100TB의 저장 용량을 갖추고 256비트 암호화를 지원합니다.

## 문제 7

서버를 확장성 있게 구성하거나 관리할 필요 없이 비즈니스 로직 코드로 서비스를 구성할 수 있는 NCP 상품은?

**보기:**
- A. Lambda
- B. Cloud Functions
- C. Serverless
- D. ActiveCode

> **정답: B. Cloud Functions**
>
> **해설:** Cloud Functions는 NCP의 서버리스(Serverless) 컴퓨팅 서비스로, 서버 관리 없이 함수 단위로 코드를 실행할 수 있습니다. AWS Lambda와 유사한 개념입니다.

## 문제 8

고객의 기존 인프라 환경과 네이버 클라우드 플랫폼을 전용선을 이용하여 사설 네트워크로 연결하는 상품은?

**보기:**
- A. Lease Line
- B. Direct Connect
- C. Active Connect
- D. Cloud Connect

> **정답: D. Cloud Connect**
>
> **해설:** Cloud Connect는 고객의 온프레미스 환경과 NCP를 전용선(Dedicated Line)으로 연결하는 서비스입니다. 안정적이고 보안성 높은 사설 네트워크 연결을 제공합니다.

## 문제 9

국내 서비스로 한국 대상 대용량 파일 다운로드를 하고자 할 때 현재 사용 가능한 적절한 상품은?

**보기:**
- A. CDN
- B. CDN+
- C. GCDN
- D. GRM

> **정답: B. CDN+**
>
> **해설:** CDN+는 국내 서비스에 최적화된 CDN 상품입니다. GCDN(Global CDN)은 해외 서비스에 적합하며, GRM은 DNS 기반 글로벌 트래픽 분산 서비스입니다.

## 문제 10

현재 네이버 클라우드 플랫폼에서 멀티존을 제공하고 있는 리전은?

**보기:**
- A. 일본
- B. 한국
- C. 독일
- D. 홍콩

> **정답: B. 한국**
>
> **해설:** NCP는 한국 리전에서 멀티존(Zone 1, Zone 2)을 제공합니다. 여러 존에 리소스를 분산하여 고가용성을 구성할 수 있습니다.

## 문제 11

다음 VPC 환경에서의 Kubernetes Service 상품에 대한 설명 중 틀린 것은?

**보기:**
- A. 클러스터 형태로 Kubernetes Service를 제공한다
- B. 최대 노드 생성 개수는 20개이다
- C. Private Subnet에서만 생성 가능하다
- D. Init Script 적용이 가능하다

> **정답: B. 최대 노드 생성 개수는 20개이다**
>
> **해설:** VPC 환경의 Kubernetes Service에서 워커 노드는 최대 50대까지 생성 가능합니다. Classic 환경에서는 최대 10대입니다.

## 문제 12

네이버 클라우드 플랫폼 VPC 환경에서 멀티존 이용이 가능한 상품은? (복수선택)

**보기:**
- A. Cloud DB MySQL
- B. Load Balancer
- C. Server
- D. Cloud Hadoop

> **정답: A. Cloud DB MySQL, B. Load Balancer**
>
> **해설:** VPC 환경에서 멀티존을 지원하는 주요 상품은 Cloud DB for MySQL과 Load Balancer입니다. 일반 서버와 Cloud Hadoop은 단일 존에서 운영됩니다.

## 문제 13

Windows SID 예방하기 위해 해야하는 작업을 바르게 설명한 것은?

**보기:**
- A. 윈도우 서버 상태를 정지 상태로 만든 후 이미지를 생성한다
- B. 윈도우 서버에 할당된 스토리지를 제거한 후 이미지를 생성한다
- C. Initialization Tool을 이용해서 sysprep 과정을 거친다
- D. 윈도우 서버에 ESC를 "사용안함"으로 변경한 후 이미지를 생성한다

> **정답: C. Initialization Tool을 이용해서 sysprep 과정을 거친다**
>
> **해설:** Windows SID(Security Identifier) 중복을 방지하기 위해 내 서버 이미지를 생성하기 전에 Initialization Tool을 실행하여 sysprep 과정을 거쳐야 합니다. sysprep은 시스템을 일반화하여 새 SID를 부여할 준비를 합니다.

- B. Init Script
- C. ACG
- D. 서버 타입

> **정답: B. Init Script**
>
> **해설:** 유사 서버 생성 시 Init Script는 반영되지 않습니다. VPC, ACG, 서버 타입 등의 설정은 기존 서버 기준으로 반영됩니다.

## 문제 17

VPC환경/Classic 2세대 서버의 Standard 서버 스펙 중 생성 가능한 서버 스펙은?

**보기:**
- A. vCPU 2개, RAM 8GB
- B. vCPU 2개, RAM 4GB
- C. vCPU 2개, RAM 16GB
- D. vCPU 4개, RAM 4GB

> **정답: A. vCPU 2개, RAM 8GB**
>
> **해설:** Standard 서버의 경우 vCPU:RAM 비율이 1:4입니다. vCPU 2개에 RAM 8GB가 Standard 타입으로 생성 가능한 스펙입니다.

네이버 클라우드 플랫폼 오토 스케일링에 대한 설명으로 옳은 것을 모두 고르시오.

- A. A 이후 생성된 스냅샷 이미지는 모두 삭제된다
- B. A 이전 생성된 스냅샷 이미지는 모두 삭제된다

## 문제 23

리눅스 서버에 추가 스토리지가 10GB 였는데 이를 20GB로 용량을 증설한 경우 기존 스토리지에 있는 데이터를 삭제하지 않고 볼륨을 확장하는 패키지는?

**보기:**
- A. LVM
- B. Spanned Volume
- C. fdisk
- D. growpart

> **정답: D. growpart**
>
> **해설:** growpart는 데이터 손실 없이 파티션 크기를 확장하는 Linux 도구입니다. Block Storage 용량 증설 후 growpart로 파티션을 확장하고, resize2fs로 파일시스템을 확장합니다.

## 문제 25

VPC 환경을 기준으로 하였을 때, Zone에 종속적이지 않은 상품은?

**보기:**
- A. 서버
- B. 로드밸런서
- C. 서브넷
- D. NAT Gateway

> **정답: B. 로드밸런서**
>
> **해설:** VPC 환경에서 Load Balancer는 Zone에 종속되지 않고 멀티존 구성이 가능합니다. 서버, 서브넷, NAT Gateway는 특정 Zone에 종속되어 생성됩니다.

- A. GCDN

## 문제 27

RAID 1(미러)에 대한 설명 중 맞는 것은?

**보기:**
- A. 단순 볼륨에 비해 쓰기 속도가 뛰어나다
- B. 단순 볼륨에 비해 읽기 속도가 뛰어나다
- C. 저장공간의 손실이 없다
- D. 패리티를 이용한 디스크 복구가 가능하다

> **정답: B. 단순 볼륨에 비해 읽기 속도가 뛰어나다**
>
> **해설:** RAID 1(미러링)은 동일한 데이터를 두 디스크에 복제하므로 읽기 시 두 디스크에서 병렬로 읽어 속도가 향상됩니다. 쓰기는 두 곳에 동시에 써야 해서 느릴 수 있으며, 저장공간 손실은 50%입니다.

## 문제 28

Archive Storage에 대한 설명으로 맞는 것은?

**보기:**
- A. 자주 read가 일어나는 데이터 저장에 최적합한 스토리지이다
- B. 데이터 최소 보관 기간은 90일로 설정되어 있다

**보기:**
- A. 서버에 추가 스토리지는 최대 15개까지 장착 가능하다
- B. 서버에는 1개의 Network Interface만 장착 가능하다
- C. 서버의 관리자 계정 패스워드를 알기 위해서는 pem key가 필요하다

- B. lvcreate > vgcreate > pvcreate > fdisk

>
