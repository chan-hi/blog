# 안녕하세요 👋

**AI × Cloud × Security × FinOps** 를 하나로 엮는 엔지니어입니다.

단순히 인프라를 운영하는 것을 넘어, **AI로 클라우드 운영 전반을 자동화**하는 것을 목표로 성장하고 있습니다.

---

## 🧭 추구하는 방향

```
AI
├── DevOps  → 배포 자동화, CI/CD, IaC
├── SecOps  → 보안 내재화, DevSecOps, 위협 자동 대응
├── FinOps  → 클라우드 비용 최적화, 이상 탐지
└── AIOps   → 장애 예측, 자동 복구, Intelligent Ops
```

> "운영의 모든 반복 작업은 자동화되어야 한다."

---

## 🛠 기술 스택

| 영역 | 기술 |
|------|------|
| **Cloud** | NCP, AWS, NHN Cloud |
| **Container** | Kubernetes, Docker, Helm |
| **IaC** | Terraform, Ansible |
| **CI/CD** | GitHub Actions, ArgoCD |
| **Observability** | Prometheus, Grafana |
| **AI/LLM** | Claude API, RAG, LLMOps |

---

## 📝 이 블로그는

실무에서 마주한 문제와 해결 과정을 기록합니다.

- 클라우드 아키텍처 설계 경험
- AI를 활용한 운영 자동화 실험
- FinOps, DevSecOps 실무 적용기
